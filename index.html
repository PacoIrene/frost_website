frost.js is a javascript library for Data visualization which depends on d3.js you can build your own charts with it

Download frost.min.js Download frost.css

  • Build Charts

    It's a good toolkit for you to build chart for you application. You can use it to build bar chart, pie chart, line chart and so on also with legend, axis and hover detail. It is a stable and reliable choice for you to build chart. See examples

  • Based on D3

    Frost.js is all based on d3 underneath, all the charts are built with SVG and css. So it is easy to use and build your own charts with all the knowledge you have known about d3. And you can also add extra functions by yourself.

  • Easy Use

    All the API is very simple and easy to use. You can only use some configs to make your chart.And also no extra codes you need to write.

  • Open Source

    Frost is free and open source, the codes are put in github and easy to see.You can download them and fork the codes in github.Hope for your pushes.
